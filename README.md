# Visualizing Housing Market Trends Using Tableau

## Project Overview

This project analyzes housing market trends using **Tableau** to understand how different housing features influence sale prices. The analysis focuses on identifying patterns between property characteristics such as **house age, renovation status, number of bedrooms, bathrooms, floors, and basement area** and their impact on housing prices.

The project uses **data visualization techniques** to present insights through interactive dashboards that help users explore housing market behavior. 

---

## Problem Statement

The objective of this project is to analyze housing market trends and determine how various housing features affect property sale prices. The goal is to identify patterns that explain variations in housing prices using visual analytics. 

---

## Dataset

The dataset used in this project was obtained from **Kaggle** and contains housing information including:

* Sale Price
* Basement Area (Sqft)
* Years Since Renovation
* Number of Bedrooms
* Number of Bathrooms
* Number of Floors
* House Age

The dataset was downloaded, saved in **CSV format**, and imported into **Tableau Desktop** for analysis. 

---

## Data Preparation

Before visualization, the dataset was cleaned and prepared to ensure accurate analysis.

### Steps performed

* Checked for missing values
* Verified correct data types
* Renamed fields for clarity
* Created calculated fields such as **House Age** and **Years Since Renovation**
* Validated dataset consistency

These steps ensured the dataset was ready for meaningful visualization and analysis. 

---

## Tableau Visualizations

Several visualizations were created to analyze the housing market trends.

### Visualizations Included

* Count of Transformed Housing Data
* Average Sale Prices
* Area of House from Basement (Sqft)
* Total Sales by Years Since Renovation
* Distribution of House Age by Renovation Status
* House Age Distribution by Bedrooms, Bathrooms, and Floors

These visualizations were combined into an interactive dashboard for better exploration of insights. 

---

## Dashboard

The Tableau dashboard provides a **comprehensive overview of housing data** by integrating multiple visualizations into a single interactive interface.

Users can easily analyze:

* Price distribution
* Impact of renovations
* Relationship between house features and prices
* Housing age trends

---

## Key Insights

The analysis revealed several important insights:

* Houses with **larger basement areas** generally have higher sale prices.
* **Renovated houses** tend to have higher market value.
* Houses with **more bedrooms and bathrooms** often sell for higher prices.
* **House age** also impacts price distribution in the market.
